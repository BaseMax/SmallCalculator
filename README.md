# Small Calculator

A small calculator to parse and execute mathematical expressions based on C, Flex, Bison.

- Flex: 2.6.1
- Bison (GNU Bison): 3.0.4
- GCC: gcc (Debian 6.3.0-18+deb9u1) 6.3.0 20170516

## Compile

```
$ make
```
